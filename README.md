## Getting Started

### Prerequisites

Things you need to use the software and how to install them.
* node
  [React.js](https://nodejs.org/)
  Download and install 

* npm
  ```sh
  npm install npm@latest -g
  ```
### Installation

How to install and setup your app.

1. Clone the repo
   ```sh
   git clone https://github.com/Hervepoh/react-marvel-quiz.git
   ```
2. Install NPM packages(all dependencies provide by package.json file)
   ```sh
   npm install
   ```
3. Runs the app 
  in the development mode.
   ```sh
    npm start
   ```
  in test mode(test runner in the interactive watch mode.)
      ```sh
    npm test
   ```
  in Production mode(test runner in the interactive watch mode.)
      ```sh
    npm run build
   ```
   Builds the app for production to the build folder.
    It correctly bundles React in production mode and optimizes the build for the best performance.

    The build is minified and the filenames include the hashes.
    Your app is ready to be deployed!

    See the section about deployment for more information.
